# OriginalRPL

This directory contains the standard Contiki-NG RPL implementation utilizing the MRHOF (Minimum Rank with Hysteresis Objective Function) objective function. 

**Purpose:**
Serves as the benchmark and baseline for evaluating our custom mobility-aware RPL implementations. It is used to compare metrics such as Packet Delivery Ratio (PDR) and control overhead against the newer versions under high-mobility scenarios.
